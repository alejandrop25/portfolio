##  Sobre o Projeto

Uma landing page de portfólio moderna, responsiva e minimalista, desenvolvida para apresentar meus projetos na área de desenvolvimento fullstack. 

O foco deste projeto foi criar uma interface ágil e limpa, utilizando **Tailwind CSS**.

##  Tecnologias Utilizadas

Este projeto foi construído com as seguintes tecnologias:

- **[HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)** 
- **[CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)**
- **[JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)**
- **[Tailwind CSS](https://tailwindcss.com/)**
